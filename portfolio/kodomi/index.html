Kodomi

[koh-doh-mee]

Since 2024, I have been leading the development of the Kodomi platform, working with a team of designers and developers to create an innovative no-code website builder.

About

Kodomi is a no-code website builder focused on flexibility and ease of use. It integrates UI libraries via GitHub and offers an internal marketplace where users can buy complete libraries or individual elements to fully customize their site. With features like a sitemap view and an AI-powered design assistant, Kodomi empowers users to plan and build websites without any coding skills, offering maximum customization and automation for unique web experiences.

Controls

↑ Images above

The main focus was on how to make the selected element controllers as customizable, convenient and compact as possible, while maintaining access to a variety of functions. As a designer, I decided to add the ability to drag and drop controllers to increase the flexibility of interaction. Drop-down fields for data entry were also implemented, allowing users to easily select and change the type of data, improving the overall user experience and increasing the efficiency of working with the interface.

Layers and pages

↑ Images above

The focus was on how to best organize the display of layers and pages, as well as their arrangement for the convenience of users. As a designer, I made key decisions to improve the visibility of nested layers and their states. To improve the clarity of the interface, all elements were drawn in components using boolean and text options, which made it easier to navigate and interact with the interface, improving the user experience and reducing the likelihood of errors.

Assets editor

↑ Images above

I designed the interface to simplify how users view, edit, and customize elements, ensuring consistent styling and easy access. By enabling the creation and modification of components within a unified system, I streamlined the editing process and improved overall efficiency. Additionally, I implemented seamless integration of new elements and UI libraries, ensuring flexibility for users while maintaining a cohesive design structure across portfolio.

AI designer

↑ Images above

By leveraging intelligent algorithms, I ensured that users could quickly generate custom designs for elements or entire pages based on their preferences, simplifying the design process. I focused on creating a system where AI could adapt existing designs or apply changes, all while maintaining the project's visual unity. Additionally, I integrated UI bundles, allowing the AI to automatically select and fit components seamlessly, enhancing user creativity while ensuring consistency and professional results.

Navigation

↑ Image above

An important aspect was improving the navigation bar in the editing mode, where my main goal was to reduce the number of transitions between pages, while maintaining the functionality of each page and their usability. As a designer, I made decisions aimed at ensuring easy access to key functions such as publishing, sharing, commenting and previewing, which improved the overall user experience and made working with the project more intuitive and efficient.

Sitemap view

↑ Images above

This feature allows users to define templates at the sitemap level, ensuring design consistency across all pages. I also prioritized a simple hierarchy view, making site planning more intuitive and efficient. Based on user feedback and surveys, this feature has proven to greatly enhance the site-building process by providing better control and visibility over the site's structure.

Next

Next steps for Kodomi involve further development of the marketplace for buying and selling assets, expanding customization settings, and refining the overall project functionality. This includes enhancing the AI Designer, improving the sitemap editor, integrating more third-party UI libraries, and preparing the platform for market entry. We will also focus on user testing and securing partnerships to scale Kodomi's reach and capabilities.
