About

Kodomi is a no-code website builder focused on flexibility and ease of use. It integrates UI libraries via GitHub and offers an internal marketplace where users can buy complete libraries or individual elements to fully customize their site. With features like a sitemap view and an AI-powered design assistant, Kodomi empowers users to plan and build websites without any coding skills, offering maximum customization and automation for unique web experiences.
